导出名

在 Go 中，首字母大写的名称是被导出的。在导入包之后，你只能访问包所导出的名字，任何未导出的名字是不能被包外的代码访问的。因此 Foo 和 FOO 是会被导出到作用域，foo 不会。我感觉这个有点像 Python 里面的包导出机制
